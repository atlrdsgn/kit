### Install.

To get started, add `@atlrdsgn/kit` to your dependencies.

```yarn
yarn add @atlrdsgn/kit
```

```pnpm
pnpm install --save @atlrdsgn/kit
```

```npm
npm install --save @atlrdsgn/kit
```
